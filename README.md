---

## 📥 Instruções de Instalação

Como este é um projeto em desenvolvimento no AI Studio, a forma mais segura de obter o APK atualizado é:

1.  No menu do **Google AI Studio**, procure pela opção de exportação ou **Download APK**.
2.  Aguarde a conclusão do build (isso garante que você terá a versão com todas as correções de compatibilidade).
3.  Transfira o arquivo `.apk` gerado para o seu celular.

---

## 🛠️ Correções de Compatibilidade Realizadas

Realizamos as seguintes correções críticas para garantir que o app funcione em todos os dispositivos:

1.  **Assinatura Digital (V1/V2):** Configurada para máxima compatibilidade com versões antigas e novas do Android.
2.  **SDK Atualizado (v35):** Otimizado para os modelos mais novos (Android 14 e 15).
3.  **Remoção de Restrições de Teste:** O APK agora permite a instalação direta em celulares físicos sem erros de "pacote corrompido".
4.  **Identidade Visual:** Tema **BRSolarTheme** aplicado em todo o sistema.

### **Dicas para Instalação:**
* Se o **Google Play Protect** avisar sobre "App Desconhecido", clique em **"Mais Detalhes"** e selecione **"Instalar assim mesmo"**.
* Certifique-se de desinstalar versões anteriores do app antes de instalar a nova para evitar conflitos de assinatura.
