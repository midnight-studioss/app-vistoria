# BR Solar - Sistema de Vistorias Técnicas ☀️

Este é o aplicativo profissional de vistorias técnicas para **BR Solar**, projetado para rodar diretamente em seu celular Android.

---

## 📥 Como Baixar e Instalar o APK

Para baixar o arquivo de instalação (APK) que funciona em qualquer telefone Android:

1. **Localize o arquivo:** No menu à esquerda (Explorador de Arquivos), localize o arquivo chamado **`BR_Solar_App.apk`**.
2. **Baixar:** Clique com o botão direito no arquivo **`BR_Solar_App.apk`** e selecione **"Download"**.
3. **Instalar:**
   - Transfira o arquivo para o seu celular (ou baixe diretamente se estiver acessando pelo navegador do celular).
   - Abra o arquivo no seu smartphone.
   - Se o sistema perguntar, autorize a **"Instalação de fontes desconhecidas"**.
   - Siga as instruções na tela para concluir a instalação.

---

## 🛠️ Solução de Problemas (Motorola e outros dispositivos)

Se você receber a mensagem "Não é possível instalar o app no seu dispositivo":

* **Versão do Android:** Este app foi otimizado para ser compatível com a maioria das versões atuais (Android 7.0 até Android 14/15).
* **Espaço:** Verifique se há espaço suficiente na memória do celular.
* **Play Protect:** Às vezes, o Google Play Protect pode bloquear apps baixados fora da loja. Você pode desativá-lo temporariamente nas configurações da Play Store ou clicar em "Instalar assim mesmo" (Install anyway).
* **Permissões:** Certifique-se de que o navegador ou gerenciador de arquivos tem permissão para "Instalar Apps Desconhecidos".
